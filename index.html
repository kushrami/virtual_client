virtual_client

This program is to convert our computer into a virtual client which respond based on your voice commands. you just give commands to your computer it will work upon it.

It will send E-mail, read news headlines, weather report, current time and date, open any documents any movies and musics, editing in documents, etc. also if you have your own python program and wants to run ? just add it to main file it will also execute when you will speak somethings. and plus point of this code you won't need to remember any command you can tell computer "open hulk movie" or "please open hulk movie" or "I wants to see hulk movie open it" it will open hulk movie.

i want to make this project more ahead, It is an open ended project so we can add as many as cool features we wants to add.so please join me on github i want contributor for my project.hope you like it.
